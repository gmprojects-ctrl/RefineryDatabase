# RefineryDatabase
Simple Refinery Database extracted out of Wikipedia list of Refineries https://en.wikipedia.org/wiki/List_of_oil_refineries
- Note, this is not a comprehensive list of oil refenries in the world nor are all values filled in.
- Missing values are filled in with 0 and refinery db is a work in progress.
- Plan to upload to a proper Docker SQL will come soon.


## To Run
- Create a virtual environment in python and install the requirements.txt
- Run generateCSV.py, to create a list of refineries as refinery_data.csv
